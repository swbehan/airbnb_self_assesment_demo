# Basic example loading AIRBNB Listings

## Project Objective
This project is a practice exercise to enhance my skills in Bootstrap, JavaScript, HTML, and CSS. The goal is to understand how these components interact while working with real data from a JSON file. The project simulates an Airbnb listing page, showcasing listings dynamically and styled using Bootstrap/CSS.

## Screenshot
![Airbnb Listings Screenshot](./images/air-bnb-demo-image.png)

## Tech Requirements
- **HTML5**: For structuring the content.
- **CSS3**: For styling the webpage.
- **Bootstrap 5**: For responsive design and pre-built components.
- **JavaScript (ES6)**: For dynamic content rendering and interactivity.
- **JSON**: For loading and displaying real data.
